Описание:

Проект содержит 6 смарт-контрактов:

Counter — счётчик

TaskManager — список задач

ProductStore — магазин товаров

VotingSystem — голосование

SubscriptionSystem — подписки

ProjectFunding — финансирование проектов

Среда разработки:

Remix Desktop
Solidity ^0.8.20

Как запускать:

Открыть Remix

Загрузить first_contract.sol
Скомпилировать

Deploy в Remix VM
